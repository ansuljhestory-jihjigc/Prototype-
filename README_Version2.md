# My Restaurant Website

यह एक डेमो रेस्टोरेंट वेबसाइट प्रोजेक्ट है, जिसमें About, Menu और Contact जैसे सेक्शन हैं।

## फोल्डर स्ट्रक्चर
```
my-restaurant/
├── index.html
├── README.md
├── assets/
│   ├── images/
│   └── icons/
├── css/
│   └── style.css
├── js/
│   └── script.js
├── fonts/
└── docs/
```

## कैसे शुरू करें

1. प्रोजेक्ट को डाउनलोड करें या क्लोन करें।
2. `index.html` को ब्राउज़र में खोलें।
3. अपनी images, मेनू आदि assets/sections में जोड़ें।

## कस्टमाइज़ करें

- Restaurant का नाम, about info, मेनू, contact details बदल सकते हैं।
- फोटो assets/images/ में रखें।
- नए CSS, JS features भी जोड़ सकते हैं।
